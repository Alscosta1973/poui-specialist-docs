# /poui-specialist:docs

Consulta a documentação de componentes PO-UI a partir da skill de referência embutida no plugin — sem precisar abrir o browser.

## Sintaxe

```bash
/poui-specialist:docs <nome-do-componente>
```

## O que retorna

Para cada componente, o comando retorna:

- **Descrição** — o que o componente faz e quando usar
- **Inputs** (`@Input()`) — todos os atributos com tipo, obrigatoriedade e descrição
- **Outputs** (`@Output()`) — todos os eventos com tipo do payload
- **Exemplo de uso** — código HTML e TypeScript prontos para copiar
- **Quirks** — comportamentos não óbvios documentados de produção (quando existem)

## Cobertura

O comando cobre todos os componentes PO-UI documentados na skill `poui-components`:

- **Páginas** — `po-page-list`, `po-page-edit`, `po-page-detail`, `po-page-default`
- **Tabelas** — `po-table`, `PoTableColumn`, `PoTableAction`, templates de célula/coluna/linha
- **Campos de formulário** — `po-input`, `po-select`, `po-combo`, `po-lookup`, `po-datepicker`, `po-switch`, `po-checkbox-group`, `po-radio-group`, `po-multiselect`, `po-textarea`, `po-upload`, `po-number`, `po-decimal`, `po-password`, `po-time`
- **Modal e diálogos** — `po-modal`, `PoDialogService`, `po-dynamic-view`
- **Navegação e Ações** — `po-button`, `po-menu`, `po-toolbar`, `PoBreadcrumb`, `po-tabs`, `po-button-group`, `po-dropdown`, `po-divider`
- **Layout** — `po-container`, `po-accordion`, `po-list-view`, `po-disclaimer-group`
- **Feedback e status** — `po-tag`, `po-info`, `[p-tooltip]`, `po-progress`
- **Utilitários** — `po-loading-overlay`, `po-widget`, `po-avatar`, `po-badge`, `po-skeleton`
- **UI / Visualização** — `po-chart`, `po-statistic`, `po-rich-text`, `po-calendar`, `po-code-editor`, `po-popover`, `po-key-value`, `po-context-menu`
- **Formulários dinâmicos** — `PoDynamicFormField`, `PoDynamicViewField`, `po-dynamic-form`
- **Páginas dinâmicas** (`@po-ui/ng-templates`) — `po-page-dynamic-table`, `po-page-dynamic-search`, `po-page-dynamic-edit`, `po-page-dynamic-detail`
