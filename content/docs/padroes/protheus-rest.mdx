# Padrões de Integração REST Protheus

## Contrato de Resposta Padrão

```typescript
interface ProtheusListResponse<T> {
  items: T[];
  hasNext: boolean;
  po_sync_date?: string;
}
```

## Parâmetros de Query Padrão

| Parâmetro | Tipo | Descrição |
|-----------|------|-----------|
| `page` | `number` | Número da página (começa em 1) |
| `pageSize` | `number` | Itens por página (padrão: 10, máximo: 50) |
| `q` | `string` | Busca rápida em todos os campos de texto |
| `order` | `string` | Campo de ordenação; prefixar com `-` para descendente (ex: `-nome`) |

## Padrão de Service Genérico

```typescript
import { Injectable, inject } from '@angular/core';
import { HttpClient, HttpParams } from '@angular/common/http';
import { Observable } from 'rxjs';

export interface ProtheusListResponse<T> {
  items: T[];
  hasNext: boolean;
}

export interface GetAllParams {
  page?: number;
  pageSize?: number;
  q?: string;
  order?: string;
  [key: string]: unknown;
}

@Injectable({ providedIn: 'root' })
export class PedidosService {
  private readonly http    = inject(HttpClient);
  private readonly baseUrl = '/rest/api/custom/v1/pedidos';

  getAll(params: GetAllParams = {}): Observable<ProtheusListResponse<Pedido>> {
    const httpParams = new HttpParams({ fromObject: this.cleanParams(params) });
    return this.http.get<ProtheusListResponse<Pedido>>(this.baseUrl, { params: httpParams });
  }

  getById(id: string): Observable<Pedido> {
    return this.http.get<Pedido>(`${this.baseUrl}/${id}`);
  }

  create(data: Partial<Pedido>): Observable<Pedido> {
    return this.http.post<Pedido>(this.baseUrl, data);
  }

  update(id: string, data: Partial<Pedido>): Observable<Pedido> {
    return this.http.put<Pedido>(`${this.baseUrl}/${id}`, data);
  }

  delete(id: string): Observable<void> {
    return this.http.delete<void>(`${this.baseUrl}/${id}`);
  }

  private cleanParams(params: GetAllParams): Record<string, string> {
    return Object.fromEntries(
      Object.entries(params)
        .filter(([, v]) => v !== null && v !== undefined && v !== '')
        .map(([k, v]) => [k, String(v)])
    );
  }
}
```

## Chave Composta (código + loja)

A maioria das entidades Protheus usa chave composta: `codigo` (6 chars) + `loja` (2 chars).

```typescript
// Service — chave composta no path da URL
getByKey(codigo: string, loja: string): Observable<Parceiro> {
  return this.http.get<Parceiro>(`${this.baseUrl}/${codigo}/${loja}`);
}

updateByKey(codigo: string, loja: string, data: Partial<Parceiro>): Observable<Parceiro> {
  return this.http.put<Parceiro>(`${this.baseUrl}/${codigo}/${loja}`, data);
}

deleteByKey(codigo: string, loja: string): Observable<void> {
  return this.http.delete<void>(`${this.baseUrl}/${codigo}/${loja}`);
}

// Componente de lista — navega para edição com dois parâmetros de rota
tableActions: PoTableAction[] = [
  {
    label: 'Editar',
    icon: 'po-icon-edit',
    action: (row: Parceiro) =>
      this.router.navigate([row.codigo, row.loja], { relativeTo: this.route }),
  },
];

// Rotas — edição recebe ambos os parâmetros
{ path: ':codigo/:loja', loadComponent: () => ... }

// Componente de edição — lê chave composta da rota
ngOnInit(): void {
  const { codigo, loja } = this.route.snapshot.params;
  if (codigo && loja) {
    this.codigo = codigo;
    this.loja   = loja;
    this.isEdit.set(true);
    this.loadRecord();
  }
}
```

## Tratamento de Erros do Protheus

O Protheus REST **não** retorna um `{ message }` simples. Os erros chegam assim:

```json
{
  "errorMessage": "{\"code\":\"MA0001\",\"message\":\"Registro j\\u00e1 existe\",\"detailedMessage\":\"\"}"
}
```

A string interna é JSON e o texto está codificado em URI. Use sempre este decodificador:

```typescript
// Em componentes que chamam create / update / delete
private parseProtheusError(err: unknown): string {
  try {
    const e      = err as { error?: { errorMessage?: string; message?: string } };
    const errObj = JSON.parse(e.error?.errorMessage ?? '{}') as {
      code?: string; message?: string; detailedMessage?: string;
    };
    const msg    = decodeURIComponent(escape(errObj.message ?? ''));
    const detail = errObj.detailedMessage
      ? ` — ${decodeURIComponent(escape(errObj.detailedMessage))}`
      : '';
    return `Erro ${errObj.code}: ${msg}${detail}`;
  } catch {
    return (err as { error?: { message?: string } }).error?.message
      ?? 'Erro ao processar a requisição.';
  }
}

// Uso
save(): void {
  this.service.create(this.values)
    .pipe(
      catchError((err) => {
        this.notification.error(this.parseProtheusError(err));
        return EMPTY;
      })
    )
    .subscribe(() => {
      this.notification.success('Salvo com sucesso!');
      this.router.navigate(['..'], { relativeTo: this.route });
    });
}
```

## Códigos de Status HTTP

| Status | Significado | Ação sugerida |
|--------|-------------|---------------|
| 400 | Requisição inválida / validação | Exibir mensagem decodificada do erro Protheus |
| 401 | Não autenticado | Redirecionar para login (tratar no interceptor) |
| 403 | Sem permissão | Exibir mensagem "Sem permissão para esta operação" |
| 404 | Não encontrado | Exibir "Registro não encontrado" e voltar à lista |
| 409 | Conflito / duplicidade | Exibir mensagem específica do conflito |
| 500 | Erro interno do servidor | Exibir mensagem genérica "Erro interno do servidor" |

## Configuração do Proxy (proxy.conf.json)

Em desenvolvimento local, faça o proxy de `/rest` para o AppServer do Protheus para evitar problemas de CORS:

```json
{
  "/rest": {
    "target": "http://localhost:8084",
    "secure": false,
    "changeOrigin": true,
    "logLevel": "debug"
  }
}
```

Adicionar ao script `start` do `package.json`:

```json
"start": "ng serve --proxy-config proxy.conf.json"
```

> Para configurar proxy com autenticação Basic Auth, consulte o [Guia de Integração Protheus](/docs/padroes/protheus-connect).

};

export const environment = {
  production: true,
  apiBaseUrl: '/rest',
};

// No service — usa environment para compor a URL
private readonly baseUrl = `${environment.apiBaseUrl}/api/custom/v1/pedidos`;
