PO-UI Utility Components

po-loading-overlay

Sobreposição de carregamento com texto — bloqueia a UI durante operações assíncronas. Controlado com @if (loading()) no template.

Key Inputs

InputTypeDescription
p-textstringTexto exibido abaixo do spinner
p-screen-lockbooleanBloqueia a tela inteira em vez de apenas o componente pai

Exemplos

Posicionamento: O po-loading-overlay cobre o elemento pai mais próximo com position: relative. Os componentes po-page-* já aplicam isso automaticamente. Para sobrepor um div customizado, adicione style="position: relative" nele.

po-widget

Card de KPI para dashboards — exibe título, conteúdo livre e ações opcionais. Usado no template dashboard do plugin com classes de grid responsivo.

Key Inputs

InputTypeDescription
p-titlestringTítulo do card
p-helpstringTexto do tooltip do ícone de ajuda (?)
p-type'normal' | 'danger' | 'info' | 'success' | 'warning'Variante de cor do header (default 'normal')
p-primary-labelstringRótulo do botão primário
p-secondary-labelstringRótulo do botão secundário
p-disabledbooleanDesabilita as ações do card

Key Outputs

OutputPayloadDescription
(p-primary-action)voidClique no botão primário
(p-secondary-action)voidClique no botão secundário

Exemplos

Classes de grid responsivo para po-widget

po-avatar

Exibe imagem circular de perfil com fallback para iniciais quando a imagem não carrega.

Key Inputs

InputTypeDescription
p-srcstringURL da imagem de perfil
p-size'xs' | 'sm' | 'md' | 'lg' | 'xl' | 'xxl'Tamanho do avatar (default 'md')
p-loading'eager' | 'lazy'Estratégia de carregamento da imagem (default 'eager')

Key Outputs

OutputPayloadDescription
(p-click)voidClique no avatar

Exemplos

po-badge

Contador numérico ou indicador de estado visual sobreposto a qualquer elemento.

Key Inputs

InputTypeDescription
p-valuenumberNúmero exibido — exibe "9+" automaticamente acima de 9
p-status'positive' | 'negative' | 'warning' | 'info' | 'disabled'Variante semântica de cor
p-colorstringToken de cor customizado (substitui p-status)
p-size'small' | 'medium' | 'large'Tamanho (default 'medium')

Exemplos

po-badge vs po-tag: Use po-badge para contadores numéricos sobrepostos a ícones ou como indicadores de quantidade. Use po-tag para rótulos textuais de status em listas e tabelas.

po-skeleton

Placeholder animado (shimmer) exibido enquanto dados são carregados — melhora a percepção de velocidade substituindo spinners por layout esqueleto fiel ao conteúdo real.

Key Inputs

InputTypeDescription
p-type'default' | 'lines' | 'circle'Formato do skeleton (default 'default')
p-linesnumberNúmero de linhas (quando type='lines')
p-widthstringLargura do skeleton (ex: '100%', '200px')
p-heightstringAltura do skeleton (ex: '40px')

Exemplos

PoThemeService

Troca programática de tema (light / dark / custom). Usa CSS custom properties internamente.

MétodoDescrição
changeTheme(theme: PoTheme)Aplica tema customizado
changeCurrentTheme(type: PoThemeTypeEnum)Alterna entre light/dark
getTheme(): PoThemeRetorna tema atual
