/connect

Conecta um componente Angular ao Protheus real — substitui mocks, configura proxy e gera contrato TLPP quando necessário

/poui-specialist:connect

Conecta um componente Angular gerado pelo plugin aos dados reais do Protheus REST, substituindo mocks por chamadas HTTP reais, configurando o proxy.conf.json e gerando o contrato TLPP quando o endpoint ainda não existe no backend.

Sintaxe

Exemplos

O que o comando faz

O comando executa 9 passos automáticos:

PassoO que faz
1Lê o service e component, identifica padrões mock (of(MOCK_*), delay(), interceptors)
2Exibe relatório de diagnóstico e coleta dados de conexão que faltam
3Cria ou atualiza proxy.conf.json com o target do Protheus
4Substitui o service: of(MOCK) → this.http.get(...) / this.http.post(...)
5Desativa interceptors de mock (com opção de manter para rollback)
6Gera contrato TLPP se o endpoint não existir no Protheus
7Atualiza o spec: troca mocks fixos por HttpTestingController
8Roda ng build + ng test e exibe relatório final
